# Bottle & Can Redemption POS

A phone-friendly redemption center POS preview using React + Vite. It stores data in the browser using localStorage, so no database setup is required.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal.

## Deploy to Vercel

1. Upload this folder to GitHub.
2. In Vercel, click **Add New > Project**.
3. Import the GitHub repo.
4. Framework should detect as **Vite**.
5. Click **Deploy**.

## Important data note

This app uses localStorage. Data is saved only in the browser/device being used. Use CSV exports regularly as a backup.
